Это WEB приложение, созданное по MVC шаблону.
Сайт представляет собой журнал с оценками, где:
- реализована аутентификация и авторизация с ролями admin и user
- неавторизованный пользователь может
	просматривать список учебных потоков и список учеников в потоке
- пользователь, авторизованный с ролью user может
	просматривать список учебных потоков и список учеников в потоке
	просматривать оценки ученика, к которому привязан аккаунт пользователя
- пользователь с ролью admin имеет возможность 
	просматривать список учебных потоков и список учеников в потоке
	добавлять, редактировать и удалять потоки
	добавлять, редактировать и удалять данные учеников 
	заводить данные об оценках ученика. Редактировать и удалять оценки нельзя, можно только создавать корректирующую запись

Технологии, использованные в проекте.
- ASP.NET Core MVC
- Entity Framework для организации работы с базой данных на MS SQL Server
